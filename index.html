Fantasy Hockey Helper

Link to application

Fantasy Hockey Helper was a web application I created for my Honours Project. The base of the application was created in Ruby on Rails, machine learning scripts were created in Python, and the front-end was created in HTML, CSS and JavaScript. The grade I recieved for this project was an A+. The goal of this project was to create a tool that could help users of Yahoo Fantasy Hockey with all the decisions that fantasy hockey users face. To achieve this goal the biggest task at hand was creating projections via machine learning, for the performance of every player. Once the projections were completed, I then created the web application with all the required tools for the user. To create the tools information on the user's leagues and teams was required, to do this I used ouath to connect to user's Yahoo Fantasy Sports account, and REST API calls to collect information. Some examples of tools made available to the user are draft reccomendations, free agent pick-up reccomendations, reccomendations on who to drop. and schedule comparisons, and player rankings. There was also weekly reccomendations which allowed users to see reccomendations for a specific week, which is extremely useful in the playoffs. The application was also designed as a progressive web application, allowing users to save the page to their mobile devices and have it work as mobile application.
