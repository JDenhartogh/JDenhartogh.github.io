House of Commons: Social Media Live Streamer

The Social Media Live Streamer was my first out of school exeperience at building a web application from scratch. The application was created with Node.JS, with the front-end created with HTML, CSS and JavaScript. The goal of this application was to allow Members of Parliament and their staff to be able to create live streams of House sittings on their personal social media accounts. Live streaming capabilities are very important to MP's as they allow for speeches they make in the House to easily be shared live with their constituents.

Below are a few images of the application. The application was made to be very easy to use, as a lot of MP's and MP's staff are not always technologically savvy. The application was also designed to be mobile-friendly, and works great on mobile devices.







Fantasy Hockey Helper

Link to application

Fantasy Hockey Helper was a web application I created for my Honours Project. The base of the application was created in Ruby on Rails, machine learning scripts were created in Python, and the front-end was created in HTML, CSS and JavaScript. The grade I recieved for this project was an A+. The goal of this project was to create a tool that could help users of Yahoo Fantasy Hockey with all the decisions that fantasy hockey users face. To achieve this goal the biggest task at hand was creating projections via machine learning, for the performance of every player. Once the projections were completed, I then created the web application with all the required tools for the user. To create the tools, information on the user's leagues and teams was required. To do this I used ouath to connect to user's Yahoo Fantasy Sports account, and REST API calls to collect information. Some examples of tools made available to the user are draft recommendations, free agent pick-up recommendations, recommendations on who to drop. and schedule comparisons, and player rankings. There was also weekly recommendations which allowed users to see recommendations for a specific week, which is extremely useful in the playoffs. The application was also designed as a progressive web application, allowing users to save the page to their mobile devices and have it work as mobile application.
